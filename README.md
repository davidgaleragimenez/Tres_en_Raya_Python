# ❌ Mi proyecto de Tres en Raya con Python y Tkinter ⭕

¡Hola! Te doy la bienvenida a mi repositorio. Este es el proyecto final que he desarrollado como parte de mi formación en el curso de Python de **Specialisterne España**. 

La idea principal no era solo programar el clásico juego del Tres en Raya, sino darle un enfoque muy especial: **diseñar un software que sea accesible y cómodo para personas neurodivergentes**.

### 🧩 ¿Cómo he adaptado el juego para que sea más accesible?
Al desarrollar la interfaz gráfica y las mecánicas, he pensado en detalles para evitar la sobrecarga sensorial y facilitar la concentración:

* **Paleta de colores descansada:** He utilizado un tema oscuro (Nord/Dracula) para que la pantalla no brille demasiado ni canse la vista. Los colores de las fichas (azul para las X y naranja para las O) contrastan muy bien entre sí pero sin llegar a ser estridentes.
* **Control total del sonido:** El juego incluye música de fondo para ayudar a la concentración, pero viene acompañado de una barra deslizante muy intuitiva para ajustar el volumen exacto o silenciarlo por completo con un solo botón si se prefiere jugar en calma.
* **Refuerzo visual claro:** Cuando alguien consigue hacer un tres en raya, la línea ganadora se ilumina en color verde y con símbolos claros (`⭐` o `✨`). De este modo, la victoria se entiende al instante sin necesidad de leer textos largos.
* **Tiempos de espera naturales:** La máquina tarda un pequeño instante (menos de un segundo) en responder. Así el juego se siente más humano, fluido y menos brusco entre turnos.

### 🛠️ Herramientas que he utilizado
* **Lenguaje de programación:** Python 3
* **Librerías:** `Tkinter` para crear toda la ventana y los botones, `Pygame` para gestionar de forma limpia el sistema de audio, y `Random` para dar vida a los movimientos de la IA.
