# ❌ My Tic-Tac-Toe Project with Python and Tkinter ⭕

Hi! Welcome to my repository. This is the final project I developed as part of my training in the Python course at **Specialisterne España**. 

The main idea was not just to program the classic Tic-Tac-Toe game, but to give it a very meaningful focus: **designing software that is accessible and comfortable for neurodivergent individuals**.

### 🧩 How did I adapt the game to make it more accessible?
When developing the graphical interface and mechanics, I focused on specific details to prevent sensory overload and assist with concentration:

* **Low-strain color palette:** I used a dark theme (Nord/Dracula) so the screen isn't too bright and doesn't cause eye strain. The colors of the game pieces (blue for Xs and orange for Os) contrast beautifully without being glaring or overwhelming.
* **Full audio control:** The game features background music to aid focus, but it comes with a highly intuitive volume slider to adjust the audio precisely, or mute it entirely with a single button if a quieter gaming environment is preferred.
* **Clear visual reinforcement:** When a player connects three in a row, the winning line lights up in green along with clear symbols (`⭐` or `✨`). This way, the victory is instantly understood without needing to read long text prompts.
* **Natural delay times:** The AI takes a brief moment (less than a second) to respond. This makes the gameplay feel more human, smooth, and less abrupt between turns.

### 🛠️ Tools I Used
* **Programming Language:** Python 3
* **Libraries:** `Tkinter` to build the entire window and buttons, `Pygame` to cleanly manage the audio system, and `Random` to power the AI's movements.
